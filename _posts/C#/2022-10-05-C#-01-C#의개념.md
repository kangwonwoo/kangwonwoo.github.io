# C#
- 마이크로소프트사에서 개발한 **객체지향 프로그래밍 언어**
- 프로그래밍언어는 ```CPU```(한가지 프로그램만 실행 가능)를 제어하는 방법을 배우는 것

```c#
ex) 컴퓨터

( 롬바이오스 - 가장 먼저 실행해서 연결 상태를 확인
      ↓
  하드디스크 - 저장소
      ↓
    메모리  - 메모장
      ↓
     CPU    - 메모리에 저장된 내용을 연산
      ↓
    그래픽  - CPU에서 정보를 받아옴               )
```
